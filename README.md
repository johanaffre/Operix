## Operix

Operix is a mobile-first tool to plan and trace daily work for field technical teams.

## Monorepo
- `backend/` NestJS API
- `web/` Next.js manager dashboard
- `mobile/` React Native technician app

## MVP scope
- Weekly planning for technicians
- Tickets (planned → in progress → done)
- Validation with comment + photo
- Basic history / audit trail
